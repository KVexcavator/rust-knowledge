Синтаксис методов
Методы похожи на функции: они объявляются с помощью ключевого слова fn и имени, у них могут быть параметры и возвращаемое значение, они содержат код,который выполняется, когда их вызывают из другого места. 
Mетоды отли­чаются от функций тем, что они  определяются внутри контекста структуры. Их первым параметром всегда является параметр self, представляющий эк­земпляр структуры, для которого метод вызывается.

struct Rectangle {
  width: u32,
  height: u32,
}

impl Rectangle {
  fn area(&self) -> u32 {
    self.width * self.height
  }
}
* Каждая структура может иметь несколько блоков impl
=============================
Связанные функции
разрешено определять функции в блоках impl, которые не берут self в качестве параметра

например, для создания квадрата
impl Rectangle {
  fn square(size: u32) -> Rectangle {
      Rectangle {width: size, height: size}
  }
}
что бы вызвать
let sq = Rectangle::square(3)
