Устранимые ошибки с помощью Result.
==========================
Result определяется как перечисление, имеющее два варианта — Ok и Err
enum Result<T, E> {
  Ok(T),
  Err(E),
}
=========================
Краткие формы для паники
-------------------------
unwrap
Если значение Result является вариантом Ok, то метод unwrap вернет значение внутрь Ok. Если же Result равно варианту Err, то метод unwrap вызовет макрокоманду panic!

let f = File::open("hello.txt").unwrap();
-------------------------
expect
похожий на unwrap, позволяет помимо этого выбирать сообщение об ошибке макрокоманды panic!

let f = File::open("hello.txt").expect("Не получилось открыть hello.txt");
=========================
Распространение ошибок
Когда вы пишете функцию, реализация которой вызывает что-то, что, возможно, не сработает, вместо того, чтобы обрабатывать ошибку внутри этой функции, вы можете вернуть ошибку в вызывающий код, и уже он будет решать, что делать.
