Программа реализует команду bash echo 

Доступ аргументам 
--
В rust std::env::args, проверить можно так
println!("{:?}", std::env::args());
cargo run Hello world
=> Args { inner: ["target/debug/echor", "Hello", "word!"] }
играем дальше
cargo run -n helo
=> unexpected argument '-n' found
--
есть раные либы для доступа к аргументам
здесь будет применятся clap (command-line argument parser)
```Cargo.toml
...
[dependencies]
clap = "2.33"
```
cargo build
