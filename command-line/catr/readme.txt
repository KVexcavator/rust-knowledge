Задача состоит в том, чтобы написать клон программы cat, названной так потому, что она может объединять несколько файлов в один. То есть, имея файлы a, b и c, можно выполнить команду cat a b c > all, чтобы перенаправить все строки из этих трёх файлов в файл с именем all.
Будут реализованы только опции -b|--number -nonblank и -n|--number. 
Так же будет продемонстрировано, как читать обычные файлы и STDIN, если в качестве аргумента имени файла задан дефис (-)
