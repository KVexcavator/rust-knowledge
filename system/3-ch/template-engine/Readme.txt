Building a template engine(как ruby.html.erb)
---
template engine — это компонент системного программного обеспечения, который может использоваться для генерации динамических HTML-страниц производительным способом. Он содержит комбинацию программных компонентов, включая парсеры, токенизаторы, генераторы и файлы шаблонов
---
Template syntax and design
- Литералы, например, <h1> hello world </h1>
- Переменные шаблона, заключенные в фигурные скобки {{ и }}, например, <p> {{name}} </p>
- Управление логикой с помощью тега if, например, {% if amount > 100000 %} {% endif %}
- Управление циклом с помощью тега for, например, <ul>{% for customer in customer_list}<li>{{customer.name}}</li>{% endfor %}</ul>
- Импорт контента, например, {% include "footer.html" %}
- Фильтры, например, {{name | upper}}
---
Шаблонизатор состоит из двух частей:
- Парсер
- Генератор HTML
---
Процесс:
1. Строка шаблона подается в парсер: <p> Welcome {{name}} </p>.
2. Сначала парсер определяет тип строки шаблона, что называется токенизацией. Определяет тип токенов: if-теги, for-теги и переменные шаблона. Генерирует токен типа template variable (если строка шаблона содержит статический литерал, он записывается в HTML-вывод без каких-либо изменений).
3. Затем строка шаблона разбирается на статический литерал Welcome и переменную шаблона {{name}}.
4. Выходные данные парсера (из шагов 2 и 3) передаются в HTML-генератор
5. Данные из источника данных передаются шаблонизатором в качестве контекста генератору 6. Разобранные токен и строки (из шагов 2 и 3) объединяются с контекстными данными (из шага 5) для получения строки результата, которая записывается в выходной HTML-файл
