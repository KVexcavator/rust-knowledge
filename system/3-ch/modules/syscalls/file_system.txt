fs: модуль содержит операции для работы с файловыми системами и управления ими. Структуры и методы в этом модуле работают с файлами, именованием, типами файлов, каталогами, метаданными файлов, разрешениями и перебором записей в каталоге
---
path: модуль предоставляет типы PathBuf и Path для работы с путями и их изменения
