ascii: предоставляет операции со строками и символами ASCII
---
cmp: Этот модуль содержит функции для упорядочивания и сравнения значений, а также соответствующие макросы. Например Eq, позволяет сравнивать экземпляры пользовательских структур с помощью операторов == и !=
---
fmt: содержит утилиты для форматирования и печати строк. Реализация этого трейта позволяет выводить на печать любой пользовательский тип данных с помощью макроса format!
---
hash: предоставляет функциональность для вычисления хеша объектов данных
---
iter: содержит трейт Iterator. Этот трейт может быть реализован пользовательскими типами данных для итерации по их значениям
---
num: предоставляет дополнительные типы данных для числовых операций
---
ops: содержит набор свойств, позволяющих перегружать операторы для пользовательских
типов данных. Например, свойство Add можно реализовать для пользовательской структуры, а оператор + можно использовать для сложения двух структур этого типа
