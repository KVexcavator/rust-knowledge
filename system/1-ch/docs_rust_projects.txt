Rust поставляется с инструментом Rustdoc, который может генерировать документацию для проектов Rust.
---
cм пример в integ-test-example
---
Комментарии к коду пишутся с использованием:
- // для однострочных комментариев и комментариев к документации внутри контейнера
- /* */ для многострочных комментариев 
---
Комментарии к документации пишутся с использованием двух стилей:
- /// для комментирования отдельных элементов, следующих за комментариями. 
- //!. Он используется для добавления документации к элементу.

В обоих случаях rustdoc извлекает документацию из комментариев к документации онтейнера.
export BROWSER=/usr/bin/google-chrome
$BROWSER https://google.com
cargo doc --open
---
прописать насовсем
echo 'export BROWSER=/usr/bin/google-chrome' >> ~/.bashrc
source ~/.bashrc
---
---
Написание документации в файлах разметки
---
cм пример в integ-test-example, папка doc
---
rustdoc doc/itest.md
-> doc/intest.html
