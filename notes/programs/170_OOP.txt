17. Средства объектно-ориентированного программирования
===
Языки ООП имеют некоторые общие характеристики, а именно объекты, инкапсуляцию и наследование
---
В Rust  структуры и перечисления имеют данные, а блоки impl предоставляют методы
для этих структур и перечислений.
---
В Rust  можно использо­вать ключевое слово pub, чтобы выбрать, какие модули, типы, функции и методы в коде должны быть публичными
---
В Rust невозможно определить структуру, которая наследует поля и реализации методов родительской структуры
Вме­сто этого можно делиться кодом Rust с помощью реализаций по умолчанию для типажных методов
===
Использование типажных объектов,допускающих значения разных типов
Пример, библиотека gui для отслеживания большоно числа значений разных типов и вызыва метода draw для каждого из этих по-разному типизированных значений
---
Можно переделывать в типажные объекты только объектно-безопасные типа­жи.
Типаж объектно-безопасен, если все методы, определенные в нем, имеют следующие свойства:
- Возвращаемый тип не является Self.
- Параметры обобщенных типов отсутствуют.
Пример ипажа, методы которого не являются объектно-безопасными:
pub trait Clone {
    fn clone(&self) -> Self;
}
